# Aletheia Frontend UI

A clean standalone React + Vite frontend based on the `Kitibwa0776/proojec` repository theme.

## What is included

- Login page
- Register page
- Forgot password page
- Dashboard
- Projects listing
- Project details page
- Upload report page
- Compliance page
- Analytics page
- Team page
- Notifications page
- Profile page
- Settings page
- Help center page
- Admin page
- Not found page
- Mock authentication and mock data
- Role-based navigation
- Protected routes

## Run locally

```bash
npm install
npm run dev
```

## Notes

- This frontend is intentionally **not connected to the backend**.
- Data is mocked and persisted in local storage where needed.
- Styling is plain CSS for easy editing.
